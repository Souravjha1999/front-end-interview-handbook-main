---
title: Qual é a diferença entre objetos do host e objetos nativos?
---

Objetos nativos são objetos que são parte do idioma JavaScript definido pela especificação ECMAScript como `String`, `Math`, `RegExp`, `Object`, `Function`, etc.

Objetos de host são fornecidos pelo ambiente de execução (navegador ou Node), como `window`, `XMLHTTPRequest`, etc.
