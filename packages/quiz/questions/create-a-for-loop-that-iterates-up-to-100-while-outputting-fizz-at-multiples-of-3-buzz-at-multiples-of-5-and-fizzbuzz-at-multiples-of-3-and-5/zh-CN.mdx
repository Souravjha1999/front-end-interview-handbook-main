title: 创建一个for循环，迭代到`100`，同时在`3`的倍数上输出**"fizz "**，在`5`的倍数上输出**"buzz"**，在`3`和`5`的公倍数上输出**"fizzbuzz"**。
---

看看这个版本的 FizzBuzz，作者是 [Paul Irish](https://gist.github.com/jaysonrowe/1592432#gistcomment-790724)。

```js
for (let i = 1; i <= 100; i++) {
  let f = i % 3 == 0,
    b = i % 5 == 0;
  console.log(f ? (b ? 'FizzBuzz' : 'Fizz') : b ? 'Buzz' : i);
}
```
